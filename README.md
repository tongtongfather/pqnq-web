# PQNQ 레스토랑 - 미니멀 키친

pqnq.kr 도메인을 위한 미니멀하고 예술적인 레스토랑 랜딩 페이지입니다.

## 🚀 배포 정보

- **Vercel**: 정적 웹사이트 호스팅
- **GitHub**: 소스 코드 관리
- **도메인**: pqnq.kr

## 📁 프로젝트 구조

```
pqnq-restaurant/
├── index.html      # 메인 HTML 파일
├── styles.css      # 스타일시트
├── script.js       # JavaScript 로직
├── vercel.json     # Vercel 배포 설정
└── README.md       # 프로젝트 설명
```

## 🎨 주요 기능

- 미니멀리즘 기반 디자인
- 완벽한 반응형 레이아웃
- 부드러운 스크롤 애니메이션
- 모바일 최적화
